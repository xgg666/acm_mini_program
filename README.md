###环境
  - 后端：openjdk 10.0.1
  - 后台前端：node10.15.3
  - 数据库: 
    - redis-cli 4.0.9
    - mysql  Ver 14.14 Distrib 5.7.21, for Linux (x86_64) using  EditLine wrapper
  - 小程序端：微信官方小程序开发工具
###数据库导入
  - 使用mysq导入 `acm.sql`

###后端启动
  - 使用IDEA打开`rear_end`, 运行即可

###后台启动
  - 进入`back_stage/my-app2` ，运行 `npm start`, 如缺少环境，运行`npm install`

###小程序端
  - 使用开发工具打开文件夹`mini_program`编译即可
